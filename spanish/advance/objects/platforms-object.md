# Platforms

Es un array utilizado dentro de un módulo, un tema o un servicio, que indica para que plataformas están disponibles; pueden ser `android`, `ios` o `windows`.


´´´json
{
  "platforms": [
    "android",
    "ios",
    "windows"
  ]
}
´´´
