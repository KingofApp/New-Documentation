# Platforms

Es un array donde indicaremos para qué plataformas está disponible dicho módulo, tema o servicio, pueden ser `android`, `ios` o `windows`.


´´´json
{
  "platforms": [
    "android",
    "ios",
    "windows"
  ]
}
´´´
